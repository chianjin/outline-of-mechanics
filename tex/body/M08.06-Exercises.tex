\exercise 一喷气式飞机的质量为$ M $，它的发动机把空气吸进来又
由机尾喷出去。喷出的气流对于飞机的速度是不变的，等于$ u $；每
秒钟喷出去的气的质量也是不变的，等于$ m $。设各处的摩擦力都
可略去，若在$ t = 0 $时飞机由静止出发并保持水平飞行，试求：

(1)飞机的速度$ v $与时间的关系；

(2)飞机的瞬时效率$ \eta $与$ u $和$ v $的关系，何时最大？最大
值$ \eta _ { \text { max } } $等于多少？
